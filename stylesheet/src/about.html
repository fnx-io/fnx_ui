About

fnx_ui is a set of Desktop first Angular2.Dart components.

You are looking at the catalogue of all designed components. Some of them are decorated with simple jQuery JavaScript, but most of the logic will be created later in Angular2.Dart without any JavaScript at all. Please, don't be surprised if you stumble upon a date input with opened date picker and such. We are testing CSS here, not behaviour.

Move your mouse over (or tap on) an example and then click on the red code button to view example source code.
